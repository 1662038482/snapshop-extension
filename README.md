# Snapshop for VSCode

Snapshop 是一款现代、快速、跨平台的 Web 图色处理应用。

Snapshop for VSCode 是 Snapshop 的 VSCode 实现。

## 特性

-   ✨ 使用 Vue3 构建
-   ✨ 使用 Antdv3 组件
-   ✔ 纯 Web 实现
-   ✔ 支持跨平台运行环境
-   ⚡ 图片预览、取点色、取范围
-   ⚡ 流程式，可视化生成目标代码
-   ⚡ 图片对比
-   ⚡ 字库生成

## 开始使用

-   请使用 **<kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>P</kbd>** 调出命令面板，输入 `Snapshop` 打开应用。
